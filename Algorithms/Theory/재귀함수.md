# 재귀함수(Recursion Function)

재귀함수(Recursion Function)란 자기 자신을 다시 호출하는 함수를 말합니다.

## 예시
```python
def recursive_function:
  print('재귀 함수 호출')
  rescursive_function()
  
recursive_function()
```

재귀함수는 자기 자신을 다시 호출하기 때문에 함수안에 종료 조건을 명시하지 않으면 무한으로 호출하게 됩니다. Python의 경우 일정이상 출력하게 되면 **RecursionError: maximun recursion depth exceeded while pickling an object**라는 오류 메세지를 출력하면서 실행이 멈추게 됩니다.

## 대표적인 재귀함수 예제(팩토리얼)
```python
def factorial(n):
  if n <= 1:
    return 1
  return n * factorial(n - 1)
 
print(factorial(5)) // 120
```
