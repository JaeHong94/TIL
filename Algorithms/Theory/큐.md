# 큐(Queue)

큐(Queue)는 먼저 들어온 것이 먼저 나가는 구조를 말합니다. 이를 선입선출(FIFO : First In First Out)이라고 부릅니다.

## 큐에서 사용하는 메서드

- JavaScript
  - shift(): 리스트 첫번째 원소를 제거합니다.
  - push(value): 리스트 가장 마지막에 원소를 삽입합니다.
- Python(단, Python은 큐를 구현하기 위해 deque 라이브러리를 사용해야합니다.)
  - popleft(): 리스트 첫번째 원소를 제거합니다.
  - append(value): 리스트 첫번째 위치에 원소를 삽입합니다.
