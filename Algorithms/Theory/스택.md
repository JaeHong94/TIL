# 스택(Stack)

스택이란 먼저 들어온 것이 나중에 나가고, 가장 나중에 들어온 것이 가장 빨리 나가는 구조를 말합니다. 이를 선입후출(First In Last Out) 또는 후입선출(Last In First Out) 이라고 부릅니다.

## 스택에서 사용하는 메서드

- JavaScript
  - pop(): 리스트 가장 마지막 원소를 제거합니다.
  - push(value): 리스트 가장 마지막에 원소를 삽입합니다.
- Python
  - pop(): 리스트 가장 마지막 원소를 제거합니다.
  - append(value): 리스트 가장 마지막에 원소를 삽입합니다.
